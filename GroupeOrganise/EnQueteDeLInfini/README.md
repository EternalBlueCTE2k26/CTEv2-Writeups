## Informations du challenge

| Catégorie | Difficulté | Points | Auteur |
|-----------|------------|--------|--------|
| Misc | Moyen | 350 | Vaskange |

**Preuve :** `500k€`

---

## Résumé

Ce challenge nécessite de trouver le site vaskange.world et de faire l'enquête dans la quête pour retrouver la transaction d'échange entre les `FakeBrokeurs` et les hackeurs qui ont vendu le fichier clients/adhérents de la société **Camaïeu** après qu'elle a subi une cyberattaque :

1. Regarder l'activité de **Miguel SANTOS** sur les réseaux sociaux (notamment son compte `Instagram`)
2. Entamer sur le site `vaskange.world` la quête en rapport avec notre enquête

---

## Partie 1 : Identification du point d'entrée

Pour démarrer le challenge, il ne faut négliger aucun mot dans l'énoncé. Le titre `En quête de l'infini` donne quelques indications sur la nature du challenge **Misc**.

Pour trouver le point d'entrée de ce challenge, il faut considérer l'enquête dans son ensemble. Les FakeBrokeurs doivent nécessairement prendre contact avec les revendeurs de données.

Un rapide 360 sur les réseaux sociaux de nos méchants protagonistes (dont Miguel est le représentant France) nous permet de repérer, parmi les photos/vidéos sur le compte Instagram de Miguel, une publication intéressante : `https://www.instagram.com/miguel.100tos/`
